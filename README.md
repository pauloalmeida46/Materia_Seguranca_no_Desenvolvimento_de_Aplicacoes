# Segurança no Desenvolvimento de Aplicações

## 📌 Sobre o Repositório

Este repositório foi criado com o objetivo de centralizar e organizar as atividades desenvolvidas na disciplina **Segurança no Desenvolvimento de Aplicações**.

A proposta principal é manter um histórico estruturado dos exercícios, práticas e experimentos realizados ao longo da matéria, permitindo não apenas o acompanhamento da evolução do aprendizado, mas também a reutilização futura desses conteúdos como material de estudo e consulta.

## 🎯 Objetivos

- Armazenar atividades práticas relacionadas à segurança no desenvolvimento
- Documentar soluções e abordagens utilizadas em cada exercício
- Servir como material de apoio para revisões e estudos futuros
- Consolidar conhecimentos sobre boas práticas de segurança em aplicações

## 📂 Conteúdo

O repositório pode incluir:

- Scripts e comandos (ex: UFW, iptables, etc.)
- Exercícios resolvidos
- Anotações técnicas
